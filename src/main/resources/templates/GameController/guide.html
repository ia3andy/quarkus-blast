Guide

Quarkus Blast is a game to find how far we can go with Quarkus server side rendering (Renarde and Qute) and htmx.

  • You have 6 Quarks types (up, down, charm, stange, top, bottom) placed on a grid with different charges.
  • Tap on a Quark to selected it, then tap on another to swap them.
  • Tap on the same Quark to blast it
  • When the same Quarks are next to each other, a blast reaction is triggered and generate the total charge time the number of Quarks of the reaction points
  • Two reactions can happen at the same time, it's a combo

Go blast it all!
